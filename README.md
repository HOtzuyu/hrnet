## Lancement de l'application

- Clone the repo on your computer.

```bash
clone [the repository](https://github.com/HOtzuyu/hrnet.git)
```

- install the repo

```bash
# with NPM
npm install
# with Yarn
yarn
```

- launch the project

```bash
# with NPM
npm start
# with Yarn
yarn start
```
